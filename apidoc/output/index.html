    Sensors data

    • All

      no implementation for this version
      Listing of all records for a specific address.
      Request:
      Resource:
      http://178.62.253.111/live?all={address}
      URI Parameters:
      Parameter Type Optional Description
      address string Sensor address
      Response:
      Body:
      {
      "_id": {
      "oid": string Unique index
      },
      "address": string, Sensor Address identifier
      "domain": string, Sensor Domain identifier
      "property": string, Sensor Property identifier
      "timestamp": {
      "date": integer Millis timestamp
      },
      "value": integer Value
      }
      Sample:
      Request:
      GET /live?all=N105S3 HTTP/1.1 Content-Type: application/json; charset=utf-8
      Response:
      HTTP/1.1 200 OK Content-Type: application/json; charset=utf-8 { "_id": { "oid": "my_oid" }, "address": "my_address", "domain": "my_domain", "property": "my_property", "timestamp": { "date": 42 }, "value": 42 }

      "value": integer Value
